# Pharmacy Courses Website

A static website built with HTML, CSS, and JavaScript to share notes and Q&A for five pharmacy courses:
- Physiology
- Pharmaceutical Chemistry
- Anatomy
- Biochemistry
- Pharmaceuticals

## How to View
Visit the live site here: https://your-username.github.io/pharmacy-courses-site

## How to Use
- Browse each course page for notes and questions.
- Use the navigation links to switch between subjects.
